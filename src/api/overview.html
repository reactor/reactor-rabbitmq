This document is the API specification for the Reactor RabbitMQ library.

Reactor RabbitMQ is a reactive API for Apache Kafka based on Project Reactor. Reactor RabbitMQ API enables messages to be published to RabbitMQ exchanges and consumed from RabbitMQ queues using functional APIs with non-blocking back-pressure and very low overheads. This enables applications using Reactor to use RabbitMQ as a message bus or streaming platform and integrate with other systems to provide an end-to-end reactive pipeline.

Sender

    sender.send(Flux.range(0, 10).map(i -> new OutboundMessage(exchange, routingKey, ("Message_" + i).getBytes()));
          .subscribe();

Receiver

    receiver.consumeNoAck(queue)
            .subscribe(r -> log.info("Received message {} ", r));

More detailed documentation is available in the Reference Guide.
